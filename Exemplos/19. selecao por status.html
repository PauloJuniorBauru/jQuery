Selecionando elementos pelo status

São elementos do jQuery que permitem selecionar determinado elementos de acordo com seu estado, tais como: marcado, selecionado, desabilitado e com o foco do usuário.

01. Seleciona elemento marcado.
Este seletor permite localizar apenas os elementos que foram ou estão marcados.

Ex: $("input:checked)")

02. Seleciona elemento que está selecionado.
Por exemplo, se quisermos encontrar a opção de um select que foi selecionada pelo usuário, poderiamos definir o seguinte seletor.

Ex: $("option:selected)")

03. Seleciona elemento que possui o foco.
No exemplo, procuramos apenas a input que possui o foco do usuário, isto é, aquela que tem o cursor de tela no momento atual.

Ex: $("input:focus)")

04. Seleciona elemento que está desabilitado.
No exemplo, selecionamos apenas as inputs que estão desabilitadas.

Ex: $("input:disabled)")

05. Seleciona elementos visíveis.
Neste exemplo, definimos que apenas as inputs visíveis devem ser selecionadas.

Ex: $("input:visible)")

06. Seleciona elementos escondidos.
No exemplo, buscamos apenas as inputs que estão ocultas.

Ex: $("input:hidden)")

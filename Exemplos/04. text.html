Função text()

Esta função permite pegar o texto dentro de uma tag ou colocar um outro texto dentro da mesma. Abaixo temos um botão que copia o texto do segundo parágrafo para o primeiro.
Porém, preste atneção no resultado final dessa cópia. Se observarmos bem, pode-se ver que no parágrafo 2 temos uma palavra em negreito (Lorem ipsum). Quando o conteúdo é copiado, a palavra em negrito, não é copiada, apenas o seu texto. Essa é a principal característica da função text(), copiar apenas o texto, retirando todo o código html.
